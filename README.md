# Descrição: 
O jogo desenvolvido é um endless runner, constituído por 2 cenas, a cena de main menu e a cena principal do jogo.

O main menu apresenta as opções de "Play" e "Quit", em que play transporta o jogador para a cena do jogo e quit o faz sair do jogo.

Durante o jogo, o jogador pode mover o personagem para a direita e para a esquerda e também saltar. Ele deve tentar evitar os obstáculos que vão aparecendo, caso contrário receberá uma mensagem de "Game Over". Ao mesmo tempo, deve tentar apanhar as moedas para aumentar o seu "Score", que é calculado conforme a distância e total de moedas apanhadas.

Enquanto percorre o caminho, o jogador pode deparar-se com alguns "Power Ups", estes são de 4 tipos:
- Speed, duplica a velocidade;
- Jump, duplica o tamanho do salto;
- Coin Value, multiplica o valor de todas as moedas por 5;
- Shield, protege o jogador de 1 obstáculo.

# Jogabilidade:
- Mover para a esquerda - Seta esquerda, ou "A";
- Mover para a direita - Seta direita, ou "D";
- Saltar - Espaço.

# Como abrir o projeto:

1. Clonar o repositório;
2. Abrir o Unity Hub;
3. Adicionar o projeto (escolher o repositório clonado);
4. Abrir o projeto;
5. Selecionar a cena "Main Menu" em Assets -> Scenes;
6. Clicar no botão de iniciar;
